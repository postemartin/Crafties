# Team Crafties Next Steps

## Current public site
- Live at https://teamcrafties.com
- Free starter pattern library is ready.
- Community areas are clearly labeled as safe demos/previews.
- Private Mother’s Day content has been removed from the public publish folder.

## Recommended next build order

1. **Gallery submissions first**
   - Add a real upload/intake path for one finished-project photo plus a kind project note.
   - Keep submissions private until approved.
   - Start with trusted makers only.

2. **Moderation + kindness rules**
   - Add a small review queue.
   - Approve, hide, or edit only after checking consent and tone.
   - Publish clear upload guidelines before public accounts.

3. **Simple member accounts**
   - Password-free email magic link.
   - Maker name and optional craft interests.
   - Notifications only after the safety flow is clear.

4. **Comments / help room**
   - Start with question-and-answer support.
   - Add comments only after moderation is reliable.

5. **Artist of the Day**
   - Use approved gallery submissions.
   - Ask permission before featuring a maker.

## Tone guardrails
- Beginner-friendly before fancy.
- Free patterns and encouragement before selling.
- Kind comments only.
- Demo features must stay clearly labeled until they are real.
- Nath’s public voice should feel warm, gentle, practical, and proud of imperfect handmade attempts.
